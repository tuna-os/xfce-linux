# XFCE Linux Roadmap

**Last updated**: 2026-06-13 | **Status**: Alpha

Part of the [TunaOS](https://tunaos.org) ecosystem. Lightweight XFCE Wayland OCI image.

## Alpha → Beta

- **Stabilize builds** — reproducible BuildStream pipelines
- **CI pipeline** — automated builds on GitHub Actions
- **ISO publishing** — live ISO (xfce-linux-iso now merged into xfce-linux)
- **Documentation** — install guide, build instructions

## Planned

- **XFCE 4.20+** — track latest XFCE releases
- **Flatpak integration** — Flathub preconfigured
- **Hardware variants** — HWE kernel, ARM64
